Wordpress plugin that provides a configuration interface for YouTube API integration. It's meant to be used by developers who want to integrate YouTube API in their own themes/plugins.

= Links =
* [GitHub](https://github.com/massedge/wordpress-plugin-youtube-api-only)
* [Support](https://github.com/massedge/wordpress-plugin-youtube-api-only/issues)

== Installation ==

1. Download and activate the plugin through the 'Plugins' screen in WordPress.
2. Go to Settings -> YouTube API via the admin menu to access the settings page.
